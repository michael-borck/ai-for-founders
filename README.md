# AI for Founders

A founder-facing companion site for the **Curtin Ignition** AI session, the take-home so the one hour extends past the room.

> Direct it. Don't delegate. Never surrender.

## What's in it

- **A two-minute self-check** — "where are you on your AI journey?", which points the visitor at the right next step (Curious → Dabbler → Operator → Builder).
- **The RTCF method** — Role, Task, Context, Format, with a before/after worked example.
- **Starter prompts** — a founder's working set, copy-to-clipboard, each one built to make the AI argue rather than flatter.
- **Run it private** — how to run capable AI locally and free (Msty to start; Jan and GPT4All as open-source options), with the IP/data-sovereignty case.
- **A bridge** into the three-part Exec Ed series, *AI for Professionals*.

## Status

**Prototype.** Single self-contained `index.html`, no build step, no dependencies. Styled to match the rest of the estate ([bigpicture.borck.dev](https://bigpicture.borck.dev)).

## Run it

Open `index.html` in a browser. That's it. To serve locally:

```
python3 -m http.server
```
